Methodology

01

Model

Logistic regression trained on LendingClub loan data (2007–2020). Features selected via L1 regularization (Lasso). Target: charged-off within 36 months.

02

Explainability

SHAP values decompose each prediction into per-feature contributions — critical for regulatory compliance under ECOA and Fair Lending laws.

03

Evaluation

AUC-ROC of 0.87 on test set. Thresholds set at 0.25 (approve) and 0.50 (decline) to balance precision and recall for the lending use case.

04

Production path

Retrain monthly on new data, monitor for feature drift, log all decisions for audit trail, deploy via REST API to core banking system.
